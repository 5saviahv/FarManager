CharTableSet

главная | структуры
Структура CharTableSet содержит набор массивов, описывающих таблицы символов FAR. Передается в функцию CharTable. 
struct CharTableSet
{
  unsigned char DecodeTable[256];
  unsigned char EncodeTable[256];
  unsigned char UpperTable[256];
  unsigned char LowerTable[256];
  char TableName[128];
};

Элементы

DecodeTable
Таблица для перекодировки из заданной кодовой страницы в OEM (DOS).
EncodeTable
Таблица для перекодировки из кодовой страницы OEM (DOS) в заданную.
UpperTable
Таблица для преобразования из нижнего регистра в верхний.
LowerTable
Таблица для преобразования из верхнего регистра в нижний.
TableName
Название кодовой страницы.
