CmdLineSelect

главная | структуры
Структура CmdLineSelect используется в функции Control для выделения или снятия выделения с текста в командной строке FAR'а, или для получения информации о выделении. 
struct CmdLineSelect
{
  int SelStart;
  int SelEnd;
};

Элементы

SelStart
Начальная позиция выделения текста
SelEnd
Конечная позиция выделения текста

Замечания

Для снятия выделения установите SelStart и SelEnd в значение -1.
Смотрите также:
структуры | TCmdLineSelect | FCTL_GETCMDLINESELECTION | FCTL_SETCMDLINESELECTION.
