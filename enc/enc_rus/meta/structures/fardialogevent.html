FarDialogEvent

главная | структуры
Структура FarDialogEvent содержит информацию о событии, поступающем в функцию ProcessDialogEvent. 
struct FarDialogEvent
{
  HANDLE hDlg;
  int Msg;
  int Param1;
  LONG_PTR Param2;
  LONG_PTR Result;
};

Элементы

hDlg
Указатель на диалог
Msg
Одно из сообщений или событий
Param1
Параметр 1
Param2
Параметр 2
Result
Значение, возвращаемое обработчиком диалога.
