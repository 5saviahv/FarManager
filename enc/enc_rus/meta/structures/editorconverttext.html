EditorConvertText

главная | структуры
Структура EditorConvertText используется в функции EditorControl для конвертирования текста из OEM-кодировки в кодировку текущей кодовой страницы редактора FAR'а (и наоборот). 
struct EditorConvertText
{
  char *Text;
  int TextLength;
};

Элементы

Text
Указывает на текст для конвертирования.
TextLength
Длина текста для конвертирования.
