KeyBarTitles

главная | структуры
Структура KeyBarTitles позволяет переопределять заголовки функциональных клавиш. Массив этих структур передается в FAR функцией GetOpenPluginInfo. 
struct KeyBarTitles
{
  char *Titles[12];
  char *CtrlTitles[12];
  char *AltTitles[12];
  char *ShiftTitles[12];

  // FAR Manager >= 1.70
  char *CtrlShiftTitles[12];
  char *AltShiftTitles[12];
  char *CtrlAltTitles[12];
};

Элементы

Titles, CtrlTitles, AltTitles, ShiftTitles
Содержат адреса новых заголовков функциональных клавиш. CtrlTitles, AltTitles и ShiftTitles используются, когда нажаты клавиши <Ctrl>, <Alt> или <Shift> соответственно. Для сохранения стандартного заголовка FAR укажите в соответствующем элементе массива NULL.
CtrlShiftTitles, AltShiftTitles, CtrlAltTitles
Содержат адреса новых заголовков функциональных клавиш. CtrlShiftTitles, AltShiftTitles и CtrlAltTitles используются, когда нажаты клавиши <CtrlShift>, <AltShift> или <CtrlAlt> соответственно. Для сохранения стандартного заголовка FAR укажите в соответствующем элементе массива NULL. Доступно в FAR Manager версии от 1.70
