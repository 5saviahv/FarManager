EditorSaveFile

главная | структуры
Структура EditorSaveFile используется в функции EditorControl для сохранения редактируемого файла (команда ECTL_SAVEFILE). 
struct EditorSaveFile
{
  char FileName[NM];
  char *FileEOL;
};

Элементы

FileName
Здесь вы можете определить новое имя для сохраняемого файла. Имейте в виду, что здесь должен быть полный путь к файлу (включая директории). Если вы не желаете изменять имя редактируемого файла, то установите первый байт поля FileName равным нулю.
FileEOL
Последовательность, обозначающая конец строки. Может быть NULL (оставить исходную EOL-последовательность), \r\n или \n
