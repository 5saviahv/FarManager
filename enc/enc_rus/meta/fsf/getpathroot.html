GetPathRoot

главная | FarStandardFunctions
Функция FSF.GetPathRoot используется для получения имени корневого каталога из указанного пути. 
void WINAPI GetPathRoot(
  const char *Path,
  char *Root
);

Параметры

Path
Обрабатываемый путь.
Root
Указатель на буфер, в который будет помещено имя корневого каталога.

Возвращаемое значение

Нет.

Примечание

Помните, что Root должен иметь достаточный размер, чтобы в нем поместилась получаемая строка:
  • Локальные диски - буква диска, двоеточие, слеш - C:\
  • Reparse point (на файловой системе NTFS 5 - Windows 2000) - нечто похожее на
    \\?\Volume{be877ec2-afd6-11d4-b5e3-806d6172696f}\ или
    \??\D:\
  • UNC-путь - имя хоста и имя расшаренного ресурса - \\host\share\
