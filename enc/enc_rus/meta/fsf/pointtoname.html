PointToName

главная | FarStandardFunctions
Функция FSF.PointToName используется для получения указателя на имя файла в данном имени файла с путем. 
char *WINAPI PointToName(
  const char *Path
);

Параметры

Path
Обрабатываемое имя файла с путем.

Возвращаемое значение

Указатель на имя файла.
