LStrnicmp

главная | FarStandardFunctions
Функция FSF.LStrnicmp сравнивает части двух строк без учета регистра. Функция работает с кодировкой OEM. 
int WINAPI LStrnicmp(
  const char *Str1,
  const char *Str2,
  int Num
);

Параметры

Str1, Str2
Указатели на строки, которые вы хотите сравнить.
Num
Размер части строки для сравнения.

Возвращаемое значение

Функция возвращает:
-1 - если s1 < s2
1 - если s1 > s2
0 - если s1 == s2
