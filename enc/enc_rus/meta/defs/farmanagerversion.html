FARMANAGERVERSION

главная | типы и определения
Константа FARMANAGERVERSION определяет текущий номер версии FAR Manager и имеет следующий формат - 0xBBBBXXYY:

BBBB = номер билда (343 = 0x0157)
XX = major version (FAR 1.70 = 0x01)
YY = minor version (FAR 1.70 = 0x46)

Т.о. для FAR Manager 1.70 beta 3 build 343 эта константа будет равна: 0x01570146

Константа FARMANAGERVERSION формируется с помощью макроса MAKEFARVERSION
