Search and Replace plugins programming

Custom API

Search and Replace (S&R) - это плагин для FAR, позволяющий производить поиск и замену неких данных как в одном файле, так и во многих, а также работающий в редакторе и программе просмотра.

Информацию по API для разработки SRP-модулей (требуется файл помощи srplugins.chm) можно получить у автора плагина: Иван Синтюрин http://www.moris.ru/~spinoza. On-line версия находится по адресу: http://www.moris.ru/~spinoza/download/s_and_r/srplugins
