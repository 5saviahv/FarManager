SMatches

главная | Custom API | регэкспы
Возвращает результат последней компиляции. 
#define MATCHESNUM 0x10
typedef struct SMatches
{
  int s[MATCHESNUM];
  int e[MATCHESNUM];
  int CurMatch;
} *PMatches;

Элементы

s
массив смещений начал сопоставленных скобок
e
массив смещений концов сопоставленных скобок
CurMatch
Общее количество скобок в регулярном выражении

Замечания

Если скобку сопоставить не удалось (она была пропущена), то и s и e равны -1. Заметьте, что не обязательно CurMatch равен числу совпавших скобок - последние могут выпадать в любом порядке.
