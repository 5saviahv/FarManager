CRegExp

главная | Custom API | регэкспы
Инициализация и уничтожение объекта регулярного выражения.

Методы класса:

CRegExp();
CRegExp(char *Text);
~CRegExp();

DLL интерфейс:

PRegExp WINAPI reCreate();
PRegExp WINAPI reCreateCompile(char *text);
BOOL WINAPI reDestroy(PRegExp re);

Элементы

re
Указатель на объект для уничтожения.
Text
указатель на строку с регулярным выражением для компиляции.

Возвращаемое значение

Функции reCreate и reCreateCompile возвращают указатель на созданный объект, который потом будет использовать при вызове всех функций - или NULL в случае неудачи. reDestroy возвращает результат удаления объекта-регулярного выражения.

Замечания

Конструктор с параметром при инициализации компилирует выражение. Результат можно узнать через метод isok или же через метод geterror.
