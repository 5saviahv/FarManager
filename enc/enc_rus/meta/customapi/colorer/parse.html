CRegExp.Parse

главная | Custom API | регэкспы
Сопоставляет скомпилированное регулярное выражение целевой строке.

Метод класса:

bool Parse(char *Str, PMatches Mtch);

DLL интерфейс:

BOOL WINAPI reParse(PRegExp re, char *str, PMatches mtch);

Элементы

re
Указатель на объект
str
Целевая строка для обработки
mtch
Указатель на структуру для хранения сопоставленных пар скобок

Возвращаемое значение

Возвращает TRUE или FALSE в зависимости от успеха сопоставления.

Замечания

В этом методе некоторые параметры принимаются по умолчанию. Начало строки соответствует реальному началу строки, конец - реальному ээ... концу. Если хотите использовать расширенные параметры - используйте расширенный вариант Parse
