CRegExp.Parse

главная | Custom API | регэкспы
Сопоставляет скомпилированное регулярное выражение целевой строке, используя расширенные параметры.

Метод класса:

bool Parse(char *Str, char *Sol, char *Eol, PMatches Mtch, int Moves = -1);

DLL интерфейс:

BOOL WINAPI reParseParam(PRegExp re, char *str, char *Sol, char *Eol, PMatches mtch, BOOL moves)

Элементы

re
Указатель на объект
str
Целевая строка для обработки
Sol
Начало строки
Eol
Конец строки
mtch
Указатель на структуру для хранения сопоставленных пар скобок
moves
Принимается -1 по умолчанию - означает, что состояние NoMoves не меняется. Возможно указать иное значение - см. функцию SetNoMoves

Возвращаемое значение

Возвращает TRUE или FALSE в зависимости от успеха сопоставления.

Замечания

Если хотите использовать некоторые параметры по умолчанию - используйте короткий вариант Parse
