IsArchive

главная | Custom API | поддержка архивов
Проверка, является ли указанный файл архивом. 
BOOL WINAPI IsArchive(
  const char *Name,
  const unsigned char *Data,
  int DataSize
);

Параметры

Name
имя архива
Data
данные из архивного файла (указатель на данные размером DataSize от начала файла).
DataSize
размер переданных данных

Возвращаемое значение

TRUE, если тип архива поддерживается плагином.

Пример

Пример определения архива ZIP (ZIP.CPP)

      SFXSize=I;
      return(TRUE);
    }
  }
  return(FALSE);
}
