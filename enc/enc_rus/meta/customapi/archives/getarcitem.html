GetArcItem

главная | Custom API | поддержка архивов
Получить информацию о следующем элементе архива. Вызывается в цикле после вызова функции OpenArchive до тех пор, пока модуль возвращает значение GETARC_SUCCESS. 
int WINAPI GetArcItem(
  struct PluginPanelItem *Item,
  struct ArcItemInfo *Info
);

Параметры

Item
эта структура должна быть заполнена. (см. PluginPanelItem для детального описания полей структуры).
Info
Дополнительная информация пункта, которая по возможности должна заполняться (см. ArcItemInfo).

Возвращаемое значение

Возвращаемое значениеОписание
GETARC_EOF Достигнут конец архива.
GETARC_SUCCESS Item получен успешно.
GETARC_BROKEN "Битый" архив
GETARC_UNEXPEOF Неожиданный конец архива
GETARC_READERROR Ошибка чтения
