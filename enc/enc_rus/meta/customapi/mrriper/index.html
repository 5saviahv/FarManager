Mr.Ripper API

Custom API

Mr.Ripper - это плагин для FAR, позволяющий выдирать различные файлы из недр других файлов. Знает более 20 форматов файлов, среди которых WAV, AVI, BMP, JPG, PNG, GIF, DJVU, MP3, MOD, XM, IT, RTF и др. И более 15 пакетных файлов для многих известных игр. Поддержка форматов осуществляется с помощью плагинов второго уровня, как для отдельных форматов, так и для пакетных файлов.

Для разработки RIP-модулей требуется файл помощи ripapi.chm, который содержит информацию по API. Файл можно получить у автора плагина: Владимир Кубышев http://vovan.dankov.net/mrripper/doc_ripapi.html.
