DI_DOUBLEBOX

главная | Dialog API | Элементы диалога
Элемент диалога DI_DOUBLEBOX описывает двойную рамку.
struct FarDialogItem
{
  int Type          = DI_DOUBLEBOX
  int X1            = X1
  int Y1            = Y1
  int X2            = X2
  int Y2            = Y2
  int Focus         = 0
  int Selected      = 0
  DWORD Flags       = Flags
  int DefaultButton = 0
  char Data[512]    = заголовок рамки
};

Внимание!

  • Приведено схематичное изображение структуры; полное описание структуры см. здесь.

Флаги

Для элемента DI_DOUBLEBOX имеют значения следующие флаги:
ФлагОписание
DIF_SETCOLOR определить цвет элемента (младший байт параметра Flags должен содержать цветовой атрибут рамки)
DIF_LEFTTEXT Выравнивать заголовок рамки по левому краю.
DIF_SHOWAMPERSAND показывать в текстовой строке амперсанд не используя его для определения горячей клавиши

События

СобытиеОписание
DN_DRAWDLGITEM начало отрисовки рамки
DN_CTLCOLORDLGITEM запрос цветовых атрибутов для отрисовки; Param2: 
LoWord LoByte - текст заголовка рамки
                (COL_WARNDIALOGBOXTITLE или
                 COL_DIALOGBOXTITLE)
LoWord HiByte - выделенный текст заголовка рамки
                (highlighting text)
                (COL_WARNDIALOGHIGHLIGHTBOXTITLE или
                 COL_DIALOGHIGHLIGHTBOXTITLE)
HiWord LoByte - рамка
                (COL_WARNDIALOGBOX или COL_DIALOGBOX)
HiWord HiByte - 0 (не используется)
DN_HOTKEY горячая клавиша (Alt-Символ)
DN_MOUSECLICK "кликнули" клавишей мыши

Замечания

  1. Если элемент в массиве управляющих элементов стоит первым, то консольное окно FAR Manager будет содержать заголовок "заголовок рамки".
  2. Если координаты X1==X2 или Y1==Y2, то рисуется соответствующая линия (вертикальная или горизонтальная)
