FarListColors

главная | Dialog API
Структура FarListColors описывает цветовую схему для элементов DI_LISTBOX и DI_COMBOBOX. 
struct FarListColors
{
  DWORD  Flags;
  DWORD  Reserved;
  int    ColorCount;
  LPBYTE Colors;
};

Элементы

Flags
Флаги. Поле должно быть равно 0.
Reserved
Зарезервировано для будущего использования. Поле должно быть равно 0.
ColorCount
Количество элементов массива Colors
Colors
Байтовый массив цветовых атрибутов (цвет_фона+цвет_текста). По умолчанию задаются цветовые атрибуты:
