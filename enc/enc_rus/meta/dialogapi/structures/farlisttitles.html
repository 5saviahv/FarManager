FarListTitles

главная | структуры | Dialog API
Структура FarListTitles описывает заголовки (верхний и нижний) для элемента диалога DI_LISTBOX. 
struct FarListTitles
{
  int   TitleLen;
  char *Title;
  int   BottomLen;
  char *Bottom;
};

Элементы

TitleLen
Длина строки верхнего заголовка
Title
Строка, содержащая текст для верхнего заголовка списка. Может быть равна NULL (в этом случае верхний заголовок не выводится).
BottomLen
Длина строки нижнего заголовка
Bottom
Строка, содержащая текст для нижнего заголовка списка. Может быть равна NULL (в этом случае нижний заголовок не выводится).

Замечания

Члены структуры TitleLen и BottomLen обязательны только при получении заголовков списка (см. DM_LISTGETTITLES)
Смотрите также:
TFarListTitles, DM_LISTGETTITLES, DM_LISTSETTITLES
