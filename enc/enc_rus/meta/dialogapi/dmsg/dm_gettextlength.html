DM_GETTEXTLENGTH

Сообщения | Dialog API
Сообщение DM_GETTEXTLENGTH посылается Менеджеру Диалогов для получения размера данных, содержащихся в FarDialogItem.Data или значения поля FarDialogItem.Ptr.PtrLength.

Param1

ID элемента диалога

Param2

0

Return

Размер данных без завершающего символа конца строки '\0'.

Элементы

ЭлементОписание
Все Все элементы диалога.

Замечания

Сообщение DM_GETTEXTLENGTH в версиях FAR вплоть до 1.70 beta 3 возвращало размер данных с учетом символа конца строки.
