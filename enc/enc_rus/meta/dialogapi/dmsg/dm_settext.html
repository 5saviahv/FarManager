DM_SETTEXT

Сообщения | Dialog API
Сообщение DM_SETTEXT посылается Менеджеру диалогов для установки содержимого строк ввода или заголовков элементов диалога в новое значение.

Param1

ID требуемого элемента диалога.

Param2

Указатель на структуру FarDialogItemData. Значение этого параметра равное NULL игнорируется.

Return

Размер установленных данных без завершающего символа конца строки '\0'.

Элементы

ЭлементОписание
Все Все элементы диалога

Замечания

  1. Для DI_COMBOBOX влияет только на строку редактирования (не распространяется на DIF_DROPDOWNLIST)
  2. Сообщение DM_SETTEXT в версиях FAR вплоть до 1.70 beta 3 возвращало размер данных с учетом символа конца строки.
