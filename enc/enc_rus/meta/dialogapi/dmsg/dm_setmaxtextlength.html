DM_SETMAXTEXTLENGTH

Сообщения | Dialog API
Сообщение DM_SETMAXTEXTLENGTH посылается Менеджеру диалогов для установки максимального размера редактируемой строки.

Param1

ID элемента диалога.

Param2

Требуемый размер

Return

Предыдущее значение размера редактируемой строки или 0 в случае ошибки.

Элементы

ЭлементОписание
DI_COMBOBOX комбинированный список
DI_EDIT строка ввода
DI_PSWEDIT поле ввода пароля

Замечания

  1. Для DI_COMBOBOX влияет только на строку редактирования (не распространяется на DIF_DROPDOWNLIST)
  2. В версиях FAR вплоть до 1.70 beta 3 сообщение DM_SETMAXTEXTLENGTH именовалось как DM_SETTEXTLENGTH.
