DM_GETITEMDATA

Сообщения | Dialog API
Сообщение DM_GETITEMDATA возвращает 32-битное значение, связанное с элементом диалога. Каждый элемент диалога имеет соответствующее 32-битное значение, предназначенное для использования плагином, который создавал диалог.

Param1

ID элемента управления.

Param2

0

Return

Менеджер Диалогов возвращает 32-битное значение, ранее переданное плагином (или 0, если плагин не ассоциировал данные с этим элементом диалога).
