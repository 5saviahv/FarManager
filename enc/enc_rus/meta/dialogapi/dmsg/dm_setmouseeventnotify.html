DM_SETMOUSEEVENTNOTIFY

Сообщения | Dialog API
Сообщение DM_SETMOUSEEVENTNOTIFY позволяет управлять механизмом посылки первичного необработанного сообщения от мыши (DN_MOUSEEVENT) в диалоговую процедуру.

Param1

0 - выключить
1 - включить
-1 - получить состояние

Param2

0

Return

Предыдущее состояние.

Элементы

ЭлементОписание
Диалог сообщение затрагивает только диалог
