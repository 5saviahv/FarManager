DN_MOUSEEVENT

События | Dialog API
Событие DN_MOUSEEVENT посылается обработчику диалога до обработки ядром событий мыши.

Param1

0

Param2

Указатель на MOUSE_EVENT_RECORD.

Return

TRUE - разрешить дальнейшую обработку ядром этого события
FALSE - событие обработано плагином.

Элементы

ЭлементОписание
Диалог сообщение затрагивает только диалог

Замечания

  1. По умолчанию событие DN_MOUSEEVENT не приходит в диалоговую процедуру. Для включения/выключения посылки этого события воспользуйтесь сообщением DM_SETMOUSEEVENTNOTIFY.
  2. Не включайте без нужды данный механизм - события от мыши приходят очень часто.
  3. Плагин может модифицировать поля структуры MOUSE_EVENT_RECORD.
  4. По умолчанию для открытого DI_COMBOBOX событие DN_MOUSEEVENT в диалоговую процедуру не приходит. Воспользуйтесь сообщением DM_SETCOMBOBOXEVENT, чтобы включить режим посылки DN_MOUSEEVENT в диалоговую процедуру.
