DM_GETDLGDATA

Сообщения | Dialog API
Сообщение DM_GETDLGDATA возвращает 32-битное значение, связанное с диалогом. Каждый диалог имеет соответствующее 32-битное значение, предназначенное для использования плагином, который создавал диалог.

Param1

0

Param2

0

Return

Менеджер Диалогов возвращает 32-битное значение ранее переданное плагином.

Замечания

Первоначально здесь находится то, что передано в параметре Param функции DialogEx.
