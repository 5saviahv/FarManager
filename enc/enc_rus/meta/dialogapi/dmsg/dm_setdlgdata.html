DM_SETDLGDATA

Сообщения | Dialog API
Сообщение DM_SETDLGDATA устанавливает 32-битное значение связанное с диалогом. Каждый экземпляр диалога имеет соответствующее 32-битное значение, предназначенное для использования плагином в его собственных целях.

Param1

0

Param2

Новое 32-битное значение.

Return

Предыдущее значение.

Замечания

Первоначально здесь находится то, что передано в параметре Param функции DialogEx.
