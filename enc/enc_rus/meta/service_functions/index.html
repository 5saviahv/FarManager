Сервисные функции

главная
Следующие функции дают возможность плагину использовать меню, диалоги, языковую поддержку и прочие полезности. Указатели на эти функции плагин получает в функции SetStartupInfo.

Внимание!

  • Некоторые функции, приведенные ниже, отсутствуют в ранних версиях FAR Manager. Прочитайте замечания по структуре PluginStartupInfo и функции GetMinFarVersion для более подробной информации.

Функции общего назначения

ФункцияОписание
AdvControl дополнительные управляющие функции; работают везде: в панелях/редакторе/вьюере.
CharTable позволяет получить информацию об установленных таблицах символов
CmpName сравнивает текстовую строку (например, имя файла) с образцом (маской).
Control позволяет запрашивать различную информацию и выполнять управляющие действия.
FreeDirList освобождает память, выделенную для списка файлов функциями GetDirList и GetPluginDirList
GetDirList возвращает список файлов в указанной директории
GetMsg получить строку сообщения из языкового файла.
GetPluginDirList возвращает список файлов в указанной директории эмулируемой файловой системы
Menu показывает меню
RestoreScreen восстанавливает предварительно сохраненную область экрана
SaveScreen сохраняет область экрана
ShowHelp показывает нужный топик из hlf
Text выводит текстовую строку на экран

Специфические функции редактора

ФункцияОписание
Editor обеспечивает доступ к внутреннему редактору FAR'a
EditorControl обеспечивает доступ на низком уровне к API редактора FAR'a

Специфические функции Dialog API

ФункцияОписание
DefDlgProc вызывает стандартный обработчик диалога
Dialog показывает диалог
DialogEx расширенный диалог с функцией обработчиком
InputBox позволяет ввести строку текста
Message показывает сообщение
SendDlgMessage послать сообщение обработчику диалога.

Специфические функции программы просмотра

ФункцияОписание
Viewer обеспечивает доступ к внутреннему вьюеру FAR'a
ViewerControl позволяет запрашивать различную информацию о встроенной программе просмотра FAR и управлять ее состоянием
