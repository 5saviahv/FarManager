FreeDirList

главная | сервисные функции
Функция FreeDirList освобождает память, выделенную для списка файлов функциями GetDirList и GetPluginDirList. 
void WINAPI FreeDirList(
  const struct PluginPanelItem *PanelItem
);

Параметры

PanelItem
Адрес массива структур PluginPanelItem.

Возвращаемое значение

Нет.
