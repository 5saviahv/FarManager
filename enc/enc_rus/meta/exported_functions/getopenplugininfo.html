GetOpenPluginInfo

главная | экспортируемые функции
Функция GetOpenPluginInfo вызывается для получения информации об открытом плагине. 
void WINAPI GetOpenPluginInfo(
  HANDLE hPlugin,
  struct OpenPluginInfo *Info
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPlugin или OpenFilePlugin.
Info
Указатель на структуру типа OpenPluginInfo, которая должна быть заполнена этой функцией.

Возвращаемое значение

Нет.

Замечания

  1. Структура OpenPluginInfo, передаваемая функции GetOpenPluginInfo, уже заполнена 0. Плагин обязательно должен заполнить поле OpenPluginInfo.StructSize
  2. FAR вызывает функцию GetOpenPluginInfo достаточно часто, поэтому при проектировании плагина необходимо минимизировать "расчеты", производимые в этой функции.
