InitDialogItem

главная | структуры | дополнения
Структура InitDialogItem описывает один элемент диалога. Эта структура не является стандартной - это аналог структуры FarDialogItem. Отличие только в члене Data

Внимание!

  • Структура InitDialogItem не входит в стандартное API.
struct InitDialogItem
{
  int Type;
  int X1;
  int Y1;
  int X2;
  int Y2;
  int Focus;
  DWORD_PTR Selected;
  unsigned int Flags;
  int DefaultButton;
  char *Data;
};

Замечания

Поведение структуры InitDialogItem, как аналога FarDialogItem обусловлено следующим:
Поскольку переменная FarDialogItem.Data имеет большой размер, прямая инициализация массива FarDialogItem, может заметно увеличить потребление памяти.
Для преобразования InitDialogItem в FarDialogItem можно воспользоваться функцией InitDialogItems
