FreeVirtualFindDataW

главная | экспортируемые функции
Функция FreeVirtualFindDataW должна освободить память, выделенную функцией GetVirtualFindDataW. 
void WINAPI FreeVirtualFindDataW(
  HANDLE hPlugin,
  struct PluginPanelItem *PanelItem,
  int ItemsNumber
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPluginW или OpenFilePluginW.
PanelItem
Указывает на массив структур PluginPanelItem, ранее выделенный функцией GetVirtualFindDataW.
ItemsNumber
Количество структур PluginPanelItem в массиве PanelItem.

Возвращаемое значение

Нет.
