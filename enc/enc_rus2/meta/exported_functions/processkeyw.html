ProcessKeyW

главная | экспортируемые функции
Функция ProcessKeyW позволяет перекрывать стандартную обработку клавиш на панели плагина.
int WINAPI ProcessKeyW(
  HANDLE hPlugin,
  int Key,
  unsigned int ControlState
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPluginW или OpenFilePluginW.
Key
Виртуальный код клавиши.
К параметру Key может применяться флаг PKF_PREPROCESS.
ControlState
Состояние Shift-клавиш. Комбинация флагов из перечисления FAR_PKF_FLAGS (за исключением PKF_PREPROCESS)
Например, при нажатии Shift-F7, Key= VK_F7 и ControlState = PKF_SHIFT.

Возвращаемое значение

FALSE для дальнейшей обработки FAR.
TRUE, если плагин самостоятельно обработал клавишу.

Замечания

  • см. замечание для флага PKF_PREPROCESS.
