ProcessSynchroEventW

главная | экспортируемые функции
ProcessSynchroEventW позволяет плагину выполнить код в контексте главного потока FAR Manager.
int WINAPI ProcessSynchroEventW(
  int Event,
  void *Param
);

Параметры

Event
Тип события. Одно из следующих значений (перечисление SYNCHRO_EVENTS):
СобытиеОписание
SE_COMMONSYNCHRO Событие приходит, если плагином вызывается функция Info.AdvControl(...,ACTL_SYNCHRO,Param).
Param указывает на переданные данные в параметре Info.AdvControl(...,Param).
Возвращаемое значение должно быть 0.
Param
Значение указателя зависит от типа события. Смотрите описание событий для уточнения информации.

Возвращаемое значение

Возвращаемое значение зависит от типа события, cмотрите описание событий для уточнения информации.
Для необрабатываемых плагином событий функция должна возвращать 0.
