FarKeyToName

главная | FarStandardFunctions
Функция FSF.FarKeyToName конвертирует код клавиши, представленной во внутреннем формате FAR Manager, в строку. 
size_t WINAPI FarKeyToName(
  int Key,
  wchar_t *KeyText,
  size_t Size
);

Параметры

Key
Код клавиши во внутреннем формате FAR.
KeyText
Указатель на строку, куда будет помещён результат преобразования. Установите KeyText = NULL для того, чтобы узнать необходмый размер буфера.
Size
Максимальное количество символов, которое можно поместить в KeyText, с заключительным нулём.

Возвращаемое значение

Функция возвращает необходимый размер буфера KeyText.
