GetReparsePointInfo

главная | FarStandardFunctions
Функция FSF.GetReparsePointInfo позволяет для указанной символической ссылки или связи каталогов получить информацию об источнике - путь к исходному файлу/каталогу. 
int WINAPI GetReparsePointInfo(
  const wchar_t *Src,
  wchar_t *Dest,
  int DestSize
);

Параметры

Src
Источник данных. Строка должна содержать полное имя к символической ссылке и заканчиваться слешем ('\').
Dest
Строка назначения. Может быть равна NULL.
DestSize
Размер строки назначения Dest. Если Dest=NULL, то параметр DestSize игнорируется.

Возвращаемое значение

Реальный размер строки, размещённой в Dest или требуемый размер строки для случая, когда Dest=NULL.

0 - в случае ошибки:

  • неверный путь (выставляется системный номер ошибки ERROR_PATH_NOT_FOUND)

Примечание

  1. Адреса Src и Dest могут совпадать.
  2. Возвращаемое функцией значение не несёт никакой практической пользы для сетевых дисков.
  3. В отличии от ConvertPath, функция FSF.GetReparsePointInfo применима только для симлинков.
