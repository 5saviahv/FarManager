CloseArchive

главная | Custom API | поддержка архивов
Закрыть архив. Вызывается после последнего вызова GetArcItem. 
BOOL WINAPI CloseArchive(
  struct ArcInfo *Info
);

Параметры

Info
Дополнительная информация архива, которая по возможности должна заполняться (см. ArcInfo).

Возвращаемое значение

TRUE при успешном выполнении функции.
