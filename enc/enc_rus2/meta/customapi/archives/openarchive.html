OpenArchive

главная | Custom API | поддержка архивов
Открыть архив и подготовить его к чтению. Вызывается после успешного выполнения IsArchive . 
BOOL WINAPI OpenArchive(
  const char *Name,
  int *Type
);

Параметры

Name
имя архива
Type
если плагин поддерживает несколько типов архивов, то установите тип здесь, иначе *Type должен быть равен 0

Возвращаемое значение

TRUE при успешном завершении, FALSE в случае ошибки.

Примечание

В этой функции необходимо открыть файл, получить необходимую информацию о формате, ... В общем всё то, что потребуется для успешного наполнения виртуальной файловой системы FAR`а.
