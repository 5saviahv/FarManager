FARMACRO_KEY_EVENT

главная | типы и определения
Константа FARMACRO_KEY_EVENT определяет тип входного сообщения структуры INPUT_RECORD, которое передаётся в ProcessEditorInputW во время исполнения макрокоманды. Элемент Event содержит структуру KEY_EVENT_RECORD с информацией о сообщении с клавиатуры.

Внимание!

  • Это сообщение специфичное для FAR manager и приходит плагину во время "проигрывания" макрокоманды.
