RegExpControl

главная | сервисные функции
Функция RegExpControl позволяет управлять Регулярными выражениями. 
int WINAPI RegExpControl(
  HANDLE hHandle,
  int Command,
  LONG_PTR Param
);

Параметры

hHandle
Смотрите описание Command. (в настоящее время значение игнорируется).
Command
Может быть одним из следующих значений (перечисление FAR_REGEXP_CONTROL_COMMANDS):
КомандаОписание
RECTL_CREATE Создаёт структуры под Регулярные выражения.
hHandle должен быть установлен в значение NULL.
Param - указатель типа HANDLE, в который запишется созданный описатель Регулярных выражений.
Функция при неудаче вернёт FALSE и Param=INVALID_HANDLE_VALUE.
RECTL_FREE Освобождает память занятую при создании Регулярных выражений.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param игнорируется.
Функция возвращает TRUE.
RECTL_COMPILE Инициализирует Регулярные выражения.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param - указатель на строку с Регулярными выражениями.
Функция возвращает TRUE при удачной компиляции строки.
RECTL_OPTIMIZE Оптимизирует Регулярные выражения после инициализации, применяется после RECTL_COMPILE.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param игнорируется.
Функция возвращает TRUE при удачной оптимизации.
RECTL_MATCHEX Сопоставляет Регулярное выражение строке.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param указатель на заполненную структуру RegExpSearch.
Функция возвращает TRUE при удачной операции сопоставления.
RECTL_SEARCHEX Ищет в строке подстроки заданные Регулярным выражением.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param указатель на заполненную структуру RegExpSearch.
Функция возвращает TRUE, если удалось найти подстроки.
RECTL_BRACKETSCOUNT Количество скобок () в Регуляном выражении.
hHandle - HANDLE Регулярных выражений, созданный по команде RECTL_CREATE.
Param игнорируется.
Функция возвращает количество скобок в Регуляном выражении, при этом всё выражение целиком интерпретируется в качестве первой скобки.
Param
Указывает на Параметр команды. Смотрите описание Command.

Возвращаемое значение

Смотрите описание Command

Пример

HANDLE re;
if (Info.RegExpControl(0,RECTL_CREATE,(LONG_PTR)&re))
{
  if (Info.RegExpControl(re,RECTL_COMPILE,(LONG_PTR)L"/[0-9]+/"))
  {
    int brackets=Info.RegExpControl(re,RECTL_BRACKETSCOUNT,0);

    FILE *log=fopen("c:\\plugins.log","at");
    fprintf(log,"br: %d\n",brackets);
    fclose(log);

    if (!brackets) 
    {
      Info.RegExpControl(re,RECTL_FREE,0);
      return;
    }

    RegExpMatch *match=(RegExpMatch*)malloc(brackets*sizeof(RegExpMatch));

    RegExpSearch search={
      L"test 123 test",
      0,
      13,
      match,
      brackets,
      0
    };

    if (Info.RegExpControl(re,RECTL_SEARCHEX,(LONG_PTR)&search))
    {
      FILE *log=fopen("c:\\plugins.log","at");
      fprintf(log,"found: %d %d\n",match[0].start,match[0].end);
      fclose(log);
    }
    else
    {
      FILE *log=fopen("c:\\plugins.log","at");
      fprintf(log,"not found\n");
      fclose(log);
    }
    free(match);
  }
  Info.RegExpControl(re,RECTL_FREE,0);
}
