InfoPanelLine

главная | структуры
Структура InfoPanelLine описывает одну строку информационной панели. Массив этих структур передаётся в FAR функцией GetOpenPluginInfoW. 
struct InfoPanelLine
{
  const wchar_t *Text;
  const wchar_t *Data;
  int Separator;
};

Элементы

Text
Заголовок параметра - текст, выровненный по левому краю панели и отображаемый цветом обычного текста (COL_PANELTEXT).
Data
Данные параметра - текст, выровненный по правому краю панели и отображаемый цветом выделенной информации (COL_PANELINFOTEXT)
Separator
Если не равно нулю, показывается разделитель. Text используется как заголовок разделителя, Data игнорируется.
