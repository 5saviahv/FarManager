MacroParseResult

главная | структуры
Структура MacroParseResult используется в функции AdvControl для возврата плагину результатов проверки макропоследовательности. 
struct MacroParseResult
{
  DWORD ErrCode;
  COORD ErrPos;
  const wchar_t *ErrSrc;
};

Элементы

ErrCode
Результат проверки (перечисление FARMACROPARSEERRORCODE):
ФлагОписание
MPEC_SUCCESS Ошибок нет
MPEC_UNRECOGNIZED_KEYWORD Неизвестное ключевое слово '%s'
MPEC_UNRECOGNIZED_FUNCTION Неизвестная функция '%s'
MPEC_FUNC_PARAM Неверное количество параметров у функции '%s'
MPEC_NOT_EXPECTED_ELSE Неожиданное появление $Else
MPEC_NOT_EXPECTED_END Неожиданное появление $End
MPEC_UNEXPECTED_EOS Неожиданный конец строки
MPEC_EXPECTED_TOKEN Ожидается '%s'
MPEC_BAD_HEX_CONTROL_CHAR Неизвестный шестнадцатеричный управляющий символ
MPEC_BAD_CONTROL_CHAR Неправильный управляющий символ
MPEC_VAR_EXPECTED Переменная '%s' не найдена
MPEC_EXPR_EXPECTED Ошибка синтаксиса
MPEC_ZEROLENGTHMACRO Пустая макропоследовательность
MPEC_INTPARSERERROR Внутренняя ошибка парсера
MPEC_CONTINUE_OTL Оператор $Continue вне цикла
ErrPos
Позиция ошибки в исходном тексте.
ErrSrc
"слово" вызвавщее ошибку, может быть пустым в зависимости от кода ошибки.
