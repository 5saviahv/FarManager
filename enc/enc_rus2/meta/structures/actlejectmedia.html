ActlEjectMedia

главная | структуры
Структура ActlEjectMedia используется в функции AdvControl для извлечения диска из съёмного накопителя (CD-ROM/USB/SUBST). 
struct ActlEjectMedia
{
  DWORD Letter;
  DWORD Flags;
};

Элементы

Letter
Буква диска съёмного накопителя
Flags
Набор флагов (перечисление FAREJECTMEDIAFLAGS):
ФлагОписание
EJECT_NO_MESSAGE не выдавать сообщения об ошибках на экран
EJECT_LOAD_MEDIA попытка "загрузить/закрыть" устройство (работает только для CD-ROM)
