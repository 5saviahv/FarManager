FarList

главная | Dialog API
Структура FarList описывает список элементов FarListItem для элементов диалога DI_LISTBOX и DI_COMBOBOX. 
struct FarList
{
  int ItemsNumber;
  struct FarListItem *Items;
};

Элементы

ItemsNumber
Количество элементов списка
Items
Указатель на массив элементов списка FarListItem.
