DialogInfo

главная | Dialog API
Структура DialogInfo позволяет узнать информацию о диалоге, является параметром в DN_GETDIALOGINFO и DM_GETDIALOGINFO. 
struct DialogInfo
{
  int StructSize;
  GUID Id;
};

Элементы

StructSize
Размер структуры.
Id
Уникальный идентификатор диалога.
