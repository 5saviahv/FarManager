FarListItemData

главная | Dialog API
Структура FarListItemData описывает данные, которые будут ассоциированы с элементом списка DI_LISTBOX или DI_COMBOBOX. 
struct FarListItemData
{
  int   Index;
  int   DataSize;
  void *Data;
  DWORD Reserved;
};

Элементы

Index
Индекс элемента списка.
DataSize
Размер данных Data или 0, если предполагается ассоциировать с пунктом списка Unicide строку.
Data
Указатель на данные.
Reserved
Зарезервировано.

Замечания

Менеджер диалогов размещает данные, ассоциируемые с пунктом списка, по следующим правилам:
DataSizeData
0 Указатель на Unicode строку. Выделяется память размером wcslen(Data)+1 байт. Передаваемые данные копируются в выделенную область памяти.
<= sizeof(DWORD) Массив символов из 4-х и менее элементов или любой указатель (например, дескриптор окна HWND). Память не выделяется. Данные размещаются в локальной области элемента списка.
> sizeof(DWORD) Произвольные данные. Выделяется память размером DataSize. Передаваемые данные копируются в выделенную область памяти.
