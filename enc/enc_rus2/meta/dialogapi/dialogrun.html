DialogRun

главная | Dialog API
Функция DialogRun выводит на экран диалог созданный функцией DialogInit. 
int WINAPI DialogRun(
  HANDLE hDlg
);

Параметры

hDlg
Описатель диалога созданный функцией DialogInit.

Возвращаемое значение

Эта функция возвращает -1, если пользователь прервал диалог, или номер выбранного диалогового пункта (номер элемента массива Item).

Замечания

Эту функцию плагин можен вызывать неограниченное количество раз, разумеется между DialogInit и DialogFree.
