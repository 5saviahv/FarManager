DM_LISTSETTITLES

Сообщения | Dialog API
Сообщение DM_LISTSETTITLES посылается Менеджеру диалогов для установки заголовков (верхнего и нижнего) у элемента DI_LISTBOX.

Param1

ID требуемого элемента диалога.

Param2

Указатель на структуру типа FarListTitles, содержащую заголовки.

Return

TRUE - заголовки установлены
FALSE -элемент не является списком

Элементы

ЭлементОписание
DI_LISTBOX список

Замечания

Для установки заголовков необходимо предварительно заполнить структуру FarListTitles: 
FarListTitles ListTitle;
ListTitle.Title=L"Some title";
ListTitle.Bottom=NULL; // сбросить нижний заголовок
Info.SendDlgMessage(hDlg,DM_LISTSETTITLES,ID,(LONG_PTR)&ListTitle);
