DM_MOVEDIALOG

Сообщения | Dialog API
Сообщение DM_MOVEDIALOG посылается менеджеру диалогов для перемещения диалога.

Param1

TRUE - Param2 содержит абсолютные координаты.
FALSE - Param2 содержит относительные координаты.

Param2

Указатель на структуру COORD, в которой указаны новые координаты левого верхнего угла диалогового окна.

Return

Структура COORD, содержащая реальные координаты левого верхнего угла диалогового окна.

Замечания

Для центрирования диалога на экране заполните структуру COORD значениями -1 и установите Param1 = TRUE.

Пример

// центрировать диалог
COORD c={-1,-1};
Info.SendDlgMessage(hDlg,DM_MOVEDIALOG,TRUE,(LONG_PTR)&c);

// переместить диалог в левый верхний угол экрана в координаты 1,1.
COORD c={1,1};
Info.SendDlgMessage(hDlg,DM_MOVEDIALOG,TRUE,(LONG_PTR)&c);

// переместить диалог на 2 позиции вправо.
COORD c={2,0};
Info.SendDlgMessage(hDlg,DM_MOVEDIALOG,FALSE,(LONG_PTR)&c);
