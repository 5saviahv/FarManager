DN_LISTHOTKEY

События | Dialog API
Сообщение DN_LISTHOTKEY оповещает обработчик о том, что пользователь для смены пункта в списке (DI_COMBOBOX, DI_LISTBOX) воспользовался горячими клавишами.

Param1

ID элемента DI_COMBOBOX или DI_LISTBOX.

Param2

Позиция выбранного элемента в списке.

Return

FALSE - разрешить изменение,
TRUE - плагин сам отработал событие.

Элементы

ЭлементОписание
DI_COMBOBOX комбинированный список
DI_LISTBOX список
