DM_GETCOMBOBOXEVENT

Сообщения | Dialog API
Сообщение DM_GETCOMBOBOXEVENT возвращает режим посылки сообщений DN_KEY и DN_MOUSEEVENT в диалоговую процедуру при навигации в открытом элементе DI_COMBOBOX.

Param1

ID требуемого элемента DI_COMBOBOX.

Param2

0

Return

Состояние режима (перечисление FARCOMBOBOXEVENTTYPE).

Элементы

ЭлементОписание
DI_COMBOBOX комбинированный список
